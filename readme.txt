== Description ==

A dynamic way to display clean & responsive interactive table for your contents in a nice view.

[See Demo here](http://plugins.rmweblab.com/interactive-table/)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/interactive-table` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add category from category page /wp-admin/edit.php?post_type=interactive_table&page=it-cell-cetegory
4. Add table item here /wp-admin/edit.php?post_type=interactive_table&page=it-settings or you can import demo from plugin demo folder /plugins/interactive-table/demo-data. Data is xml format, just import using default wordpress import tools.


== Frequently Asked Questions ==

= Is is support unlimited color? =
Yes, our pro version support unlimited color and unlimited cell.
[Pro version](https://codecanyon.net/item/responsive-interactive-table/16312261?ref=RMweblab) is available [here](https://codecanyon.net/item/responsive-interactive-table/16312261?ref=RMweblab)

= How do I set different but fixed color table without hover?
It is in Settings, /wp-admin/edit.php?post_type=interactive_table&page=it-settings, Just enable "Fixed highlight"

= Can I place table Head row left or right?
Yes, this is also available in our pro version. Even you can set table head to bottom.

[Pro version](https://codecanyon.net/item/responsive-interactive-table/16312261?ref=RMweblab) is available [here](https://codecanyon.net/item/responsive-interactive-table/16312261?ref=RMweblab)
